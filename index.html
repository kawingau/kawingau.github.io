缘起

一直以来都有写字记录的习惯，最初是用日记本写日记，后来是在QQ空间、新浪博客上写点东西，偶尔穿插着用Onenote、手机的记事本等记录。自从接触计算机以来，看到了很多个人技术博客，在赞叹之余，也萌生了“我也要自己搭一个博客”的想法。

虽然想法很早以前就有，但是后来考研、毕业论文等一堆事接踵而来，这个事情就一直耽搁到最近……（说白了就是拖延症晚期）

那些遇到的“坑”

在搭建博客的过程里，我也参考了很多文章，受益颇多。除了Hexo官方有比较完备的文档，网上更有各种非常详细的搭建教程。所以这里主要记录一下我在搭建博客时遇到的坑，不再详细叙述每一个步骤。

安装

分别安装Git、Nodejs和Hexo。其中Git和Nodejs从官网下载客户端。Hexo通过命令行安装。

Hexo

在初始化Hexo博客时，在cmd或Powershell下输入 Hexo init blog ，然而：

'git'不是内部或外部命令，也不是可运行的程序或批处理文件。

尝试了几次都是这样，想起来我是很久之前安装了Git，可能没有正确设置环境变量。

打开Git Bash，用cd命令切换到博客目录下，再输入该命令，当显示：

INFO Start blogging with Hexo!

博客即初始化成功。

Localhost

在本地预览博客时，输入 hexo s 即启动预览。默认情况下，使用4000端口。但当端口被其他进程占用时，会出现网页打不开的情况。

首先，查看端口使用情况。

打开cmd，输入 netstat -ano|findstr 4000

以列表形式显示使用该端口进程的协议、本地地址、外部地址、状态和PID。

TCP    127.0.0.1:4000         0.0.0.0:0              LISTENING
TCP    127.0.0.1:4000         127.0.0.1:55290        ESTABLISHED
TCP    127.0.0.1:55290        127.0.0.1:4000         ESTABLISHED

然后，查看PID对应的具体任务：

  • 直接在任务管理器中查看

  • 通过命令查看，输入tasklist | findstr "<PID>"

    FoxitProtect.exe             10052 Services                   0     13,076 K

    此处占用端口4000的是福昕阅读器的文档保护进程。

最后，解决方法有：

  • 杀掉占用端口的对应进程
  • 使用 hexo s -p <端口号> 更改本地预览服务端口（推荐）

主题

确定博客主题

对于选择困难症来说，选定博客主题也是一件不容易的事。主题可以自己写（本人打算留作进阶时的任务），也可以借用别人的主题。

主题选择参考：

  • 官网
  • 知乎：有哪些好看的Hexo主题？

我选择的是大名鼎鼎的Next主题，根据文档修改配置文件和个性化设置。

Markdown

Markdown编辑器

“工欲善其事，必先利其器”。选择Markdown编辑器很重要的一点是可以实时预览文章，而且最好能支持尽可能多的Markdown语法。我尝试过的方法有以下两种：

安装Notepad++的Markdown插件

使用Notepad++也有一段时间了，所以一开始想直接用Notepad++写。网络上有针对Notepad++的Markdown插件，参考资料。

其中，在导入插件时按照文章中的方法屡试不灵。于是直接将对应的.dll文件粘贴到Notepad++目录下的plugin中。

虽然达到了预期效果，但很快就发现了一些问题：

  • 有些Markdown语法不支持，如任务列表。
  • 可能出现一些错误导致Notepad++异常退出。

使用Visual Studio Code

很快，我又转向了另一款编辑器，VSCode。感觉使用体验非常不错。具体如下：

  • 打开预览：Ctrl+Shift+V或在侧边栏（资源管理器）中右键正在编辑的Markdown文档，选择打开预览。

  • 设置并排窗口：在侧边栏（资源管理器）中点击正在编辑的Markdown文档，右键选择打开到侧边，再调整两窗口位置。

VSCode的默认主题是黑色的，可以通过安装扩展来更改，我安装的是Markdown Theme.

Markdown语法

Markdown语法的入门选择——Guide Guide的Mastering Markdown

特别说明的是，在Hexo中，Markdown语法的代码块表示有几种方式：

  • `code`：行内代码，code

  • 前面空四个格：代码块，不带行数

    我是代码块

  • ```code```：代码块，带行数
